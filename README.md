# 📁 FileMagic - PDF & Image Converter

> **Free, browser-based file conversion tool. Your files never leave your computer.**

![FileMagic Screenshot](https://via.placeholder.com/800x400?text=FileMagic+Converter+Preview)

## ✨ Features

- 🔄 **6 Conversion Tools**:
  - 📄 **PDF to Word** - Extract text from PDFs
  - 🖼️ **JPG to PDF** - Convert images to PDF
  - 📸 **PDF to Image** - Extract pages as PNG/JPEG
  - 📑 **PDF Merger** - Combine multiple PDFs
  - ✂️ **PDF Splitter** - Split PDF into separate files
  - 📦 **Image Compressor** - Reduce image file size

- 🔒 **100% Private** - All processing happens in your browser
- ⚡ **Fast & Free** - No uploads, no waiting, no limits
- 🤖 **AI Assistant** - Built-in help for each tool
- 📱 **Responsive** - Works on desktop and mobile

## 🛠️ How to Use

1. Open the page in your browser
2. Click on the tool you want (PDF to Word, JPG to PDF, etc.)
3. Drag & drop your file or click to browse
4. Click the convert button
5. Your converted file downloads automatically

## 📦 Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling with Bootstrap 5
- **JavaScript** - Core functionality
- **PDF.js** - PDF parsing
- **pdf-lib** - PDF creation/merging
- **FileSaver.js** - File downloads
