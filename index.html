About FileMagic

FileMagic is a free, browser-based file conversion tool that respects your privacy.

How it works

All conversions are done entirely in your browser using JavaScript. Your files are never uploaded to any server, so your data stays private and secure.

Supported formats
  • PDF → Word (.doc), Images (PNG/JPEG), Split, Merge
  • JPG/JPEG → PDF
  • Images (JPG, PNG, WebP) → Compressed
Open source

This tool is built with open-source libraries including PDF.js, pdf-lib, and FileSaver.js. Feel free to use and modify it.

100% client-side. No data upload. No tracking.
